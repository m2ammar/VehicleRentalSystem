# Vehicle Rental System

A **Java-based Vehicle Rental System** built with **JavaFX UI** and backend logic.  
This project demonstrates the use of **multi-method programming**, handling user input, file operations, and basic GUI design in Java.

---

## Description

The Vehicle Rental System allows users to:
- Book vehicles for a specified number of rental days
- Calculate rental costs automatically
- View booking details
- Demonstrates separation of **UI and backend logic**  

This project is ideal for learning **Java fundamentals**, **JavaFX GUI development**, and **multi-method programming** for practical applications.

---

## Features

- Interactive JavaFX GUI for vehicle booking
- Backend logic separated into reusable methods
- Handles input validation and basic error checking
- Easily extendable for more vehicles or features
- Simple, beginner-friendly design demonstrating Java programming fundamentals
